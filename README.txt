Studi kasus: Anime( Dari huruf A(ufa) )
Ini aplikasi cli di mana user dapat memilih opsi anime yang ingin ditonton. 
Namun selain memilih anime untuk ditonton, user juga dapat memilih agar 
anime tersebut dapat diadaptasi dan dibaca di dalam format manga.

Pattern yang digunakan:
> Strategy Pattern
    - Terdapat pada package Anime
    - Diterapkan dalam memilih strategy anime mana yang ingin ditonton

> Adapter Pattern
    - Terdapat pada package AnimeAdapter
    - Diterapkan dalam mengadaptasi anime ke dalam format manga
